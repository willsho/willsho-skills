## 1. 新旧增长公式(诊断主轴)

**旧公式(投流驱动,已失效):**
```
GMV = 流量 × 转化率 × 客单价
```
对应旧循环:**投流 → 拉新 → 冲规模 → 再投流**。在流量越来越贵、消费者信心走低的当下,这个循环边际成本飙升、确定性崩塌。

**新公式:**
```
GMV = 新品吸引力 × 老客复购率 + 品牌搜索量 × 自然转化率 + 投流增量
```
对应新循环:**用新品创造购买理由,用老客复购提供确定性。** 投流从"主引擎"退位成"增量补充"。

**怎么用来诊断:** 把品牌的增长来源往这三块上拆——
- 第一块(新品 × 复购)是**确定性基本盘**,健康品牌的增长主要来自这里。
- 第二块(品牌搜索 × 自然转化)是**心智红利**,反映用户主动来找你的程度。
- 第三块(投流增量)应该是**锦上添花**,如果一个品牌增长几乎全靠它,就是还困在旧循环里、属于"输血续命"。

宏观背景(可用于论证为什么旧公式失效):2026 年 5 月社零增速降至 -0.6%;消费者信心指数长期低于荣枯线;4 月末全国住户存款 171.65 万亿元,比 2025 全年商品服务总价值还多 22%——钱在,但不花,消费决策变得极度谨慎。

## 2. 渠道分工逻辑

**核心判断:"全域不是在所有平台开店",而是让每个平台做它最擅长的事。** 各平台的角色分工:

| 平台 | 擅长的事 | 角色 |
|---|---|---|
| 抖音 | 品类教育、制造增量 | 把不知道的人变成知道的人 |
| 小红书 | 深度种草、口碑验证 | "用户真实反馈的镜子",成分/临床/体验的前置蓄水 |
| 天猫 / 京东 | 沉淀品牌资产、承载复购 | 确定性渠道、品牌阵地 |
| 线下门店 | 品牌溢价基础设施 | 高端化、品牌升维的物理载体 |

**关键趋势:**
- **种草平台和货架平台的边界正在模糊**:抖音日搜索成交额破亿,小红书日搜索量 4.2 亿次。货架平台也能种草,种草平台也能成交。
- 多个高端品牌(CHIC JOC、之禾)在小红书/抖音"基本零投放",把它当**口碑镜子**而非种草工具——真实反馈本身就是资产。

**诊断要点:** 看品牌有没有"渠道错配"——比如把天猫当拉新主战场猛投流(应该是复购沉淀地)、或在小红书硬种草投放反而污染了口碑镜子。理想结构是每个平台扮演对的角色,而不是每个平台都想要 ROI。

## 3. 搜索结构变化

用户的搜索行为在变,这是判断品牌心智阶段的高灵敏度指标:

- **搜索动机从"找商品"变成"找答案"**:用户搜的是"高智感通勤穿搭""静奢风"这类问题,而不是具体商品。品牌要成为某个问题的答案。
- **搜索解决"买不买",推荐解决"知不知道"**:推荐流(信息流)负责让人知道你;搜索(主动)负责临门一脚的信任确认。
- **品牌词搜索占比上升 = 主动构建信任**:用户主动搜你的品牌名,说明心智已建立。这一定程度上**打破了"网红、轻奢、忠诚度"的不可能三角**。

**品类词 vs 品牌词的诊断信号:**
- 友望除螨仪**七成搜索从品类词进入** → 说明这是个新场景(床上清洁),品牌还在吃品类教育红利,机会在于把品类词流量沉淀成品牌心智。
- 鲜朗的**"鲜朗"品牌词增速 > "烘焙粮"品类词增速** → 说明品牌已经跑赢品类,心智壁垒在形成。
- 之禾的**品牌词搜索占比攀升、推荐成新客入口** → 高净值用户从"被动接收信息"转向"主动构建信任"。

## 4. 新品策略

**核心判断:新品不是战术性追流量,而是战略性输出品牌能力。**

- **"新品不是越多越好,而是越'不可比'越好"**:好新品 = 对一个细分真实需求/场景的**重新定义**,让对手没法拿来比价比参数。多 SKU 追流量是旧循环的变体。
- **新品最大的价值不是拉新,而是表达品牌对美学/理念的理解**(裘真:"新品最大的价值不是拉新,而是表达品牌对美学的理解")。
- **功效可视化 / 命名即承诺**:薇诺娜"311 屏障系列"= 神经酰胺 AP/NP + 青刺果油 + 胆甾醇 3:1:1 精准配比,把抽象功效变成可感知的具体承诺。
- **上新节奏是战略输出**:CHIC JOC 一个月上新两次,定位为"战略性输出品牌能力",不是追流量窗口。
- **场景延展**:薇诺娜把"敏感肌"标签从急救特护延展到日常护理、防晒、美白、抗老——用新品把一个标签做成一个生态。

**诊断要点:** 问品牌的新品到底在"重新定义场景"还是"补 SKU 凑数"。前者撬动新公式里的"新品吸引力 + 品牌搜索",后者只是稀释流量。

## 5. 护城河

**最重磅的发现:9 个还在增长的品牌,没有一个把"流量能力 / 投放效率"当护城河。** 答案高度一致——**供应链、用户信任、好产品**,全是需要时间积累、白牌抄不走的东西。

**三类真护城河:**

**(1) 供应链壁垒** —— 自建供应链不是为了省钱,而是建立白牌抄不走的全链路壁垒。
- 裘真:2023 年初组建自有皮料生产团队,累计投入 7000 多万,报废 60 多万尺、价值 2000 多万元皮料;自营皮革厂 + 4 家皮具工厂、团队 700 人;新员工入职都被带到工厂,让"品牌标准刻在每个人心里"。

**(2) 用户信任壁垒** —— 靠稳定和克制长出来。
- 至本:全国统一定价、全年价差极小、不用代言人、不做流量炒作,价格体系近十年稳定 → "稳定的价格体系与低营销干预一起造就的信任壁垒"。
- 裘真:"千元定价对中国品牌意味着信任的临界点"。

**(3) 好产品 / 团队积累** —— 时间和审美的复利。
- CHIC JOC:"你可以复制我们的某个单品、某场直播、某个渠道策略,但你复制不了一个团队十三年对产品的执着和积累。"
- 跨周期数据(薇诺娜):2021 年特护霜巅峰 122 万独立用户试用,2025 年仍有 120 万独立用户购买——四年几乎不衰减,就是产品力护城河的证据。

**护城河的演化(阿迪达斯):** 从"一条护城河"(全球品牌势能 + 渠道覆盖)变成"一片交叉纵横的护城水系"(再加上本土化产品创新 + 消费者直接运营能力)。单一优势不再安全,要织一张网。

**诊断要点:** 逼问品牌——"如果一个白牌明天用一半的价格抄你,你靠什么活?" 如果答案是流量/算法/投放,这就是伪护城河。

## 6. 网红到长红

**行业规律:大部分品牌在上市第三到第五年达到增长峰值,之后面临下滑。** 跨越这道坎靠的是**系统性组织能力**,不是又一次爆款。

- 跨越的关键动作:品牌自建供应链(建立白牌抄不走的全链路壁垒)、稳定的组织和团队(裘真核心团队十年未变)、从"产品特质"向"品牌人文"升级(裘真 2025 下半年的动作)。
- 网红品牌的典型死法:靠单一爆款起盘 → 第 3-5 年流量红利见顶 → 没有沉淀下供应链/信任/组织 → 增长熄火。

**诊断要点:** 先定位品牌处在上市第几年。如果接近 3-5 年且增长开始放缓,要把诊断重心放在"有没有在搭穿越周期的系统能力",而不是再找一个爆款。

## 7. 结构性机会

增长有时不来自更努力,而来自踩对了一个确定性的结构变化。文章里出现三类:

**(1) 挤出效应(价格带分化):** 之禾——消费降级里,中间品牌竞争力下降被"挤出",高端(用户受影响小)和极致性价比两端反而受益。判断品牌在价格带上是处在受益的两端,还是被挤压的中间。

**(2) 生活方式改变:** 友望——清洁家电的增长来自"一代人生活方式的改变";吸尘器的痛点从"吸力大不大"变成"倒尘杯那一刻崩溃",于是自动集尘吸尘器 2 年从 0 做到占整体吸尘器销售额近三成。新场景 = 新品类 = 新增长。统一大市场里"任何一个小品类都是一门大生意"。

**(3) 人口 / 周期结构变化:** 鲜朗——2018-2019 第一批养宠高峰的猫步入老年,未来 3-5 年老年猫数量快速增长,于是老年猫粮是一条确定性赛道。找人口结构里正在到来的确定需求。

**价值锚定(onlytree 的定位范式):** 羽衣甘蓝粉单杯 3-5 元——"比奶茶健康、比鲜榨果汁便宜、比保健品真实",是"对健康焦虑的确认""自我生活状态的低门槛投资"。用三个对照锚点把新品类的价值讲清楚。

**质价比 / 心价比的排序(裘真):** 质价比是根基,心价比(品牌溢价)是从信任里自然生长出来的,不能反过来硬凹。

**诊断要点:** 看品牌的增长故事里有没有一个结构性的"势"。纯靠运营努力顶着逆风跑很累;找到一个正在发生的结构变化,顺势而为事半功倍。
