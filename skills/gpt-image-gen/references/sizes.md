# 比例 × 分辨率 → 实际像素对照

只有当用户对成图的**具体像素**有要求时才需要查这张表（例如"我要正好 3840×2160"）。
日常使用直接用 `--size`（比例）+ `--resolution`（1k/2k/4k）即可，不必关心精确像素。

## 直接指定像素

`--size` 也可以直接传像素尺寸，例如 `--size 1881x836` 或 `--size 887x1774`，
表中任意像素值都支持。4K 支持上述全部 15 个比例。

## size = auto

传 `--size auto` 时由服务端自动选择比例，默认回退为 `1:1`。

## 图生图时的尺寸行为

走图生图（传了 `--image` / `--image-url`）时：
- 不传 `--size`：输出分辨率 = 输入参考图的分辨率
- 传了 `--size`：强制按指定尺寸出图
