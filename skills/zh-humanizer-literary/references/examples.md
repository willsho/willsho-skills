# 示例

这些示例用于校准，不对应任何真实来源原文。

## 服务介绍去 AI 味

输入：

```text
我们致力于为品牌提供全链路内容增长解决方案，通过专业的方法论和持续复盘机制，帮助品牌实现高质量增长。
```

改写：

```text
很多品牌不是没有卖点，是卖点还停在自己人的语言里。

先处理这件事：把产品、卖点和人群重新翻译成用户能复述的一句话，再决定内容怎么发、怎么验证、怎么继续改。
```

机制：

- 删掉「致力于」「全链路」「高质量增长」。
- 把抽象服务改成读者能看见的动作。
- 用「能复述的一句话」承载文采和判断。

## 公众号开头去 AI 味

输入：

```text
在当今信息爆炸的时代，越来越多的人开始意识到个人成长的重要性。如何在复杂环境中保持清晰，是每个人都需要面对的课题。
```

改写：

```text
我最近发现，很多人的混乱不是因为事情太多。

是因为每件事都只在脑子里飘着，没有被放到一张纸上。

你一写下来，就会发现：有些事不是要立刻解决，只是需要先被命名。
```

机制：

- 从泛时代开头改成一个具体观察。
- 把「清晰」变成「一张纸」和「命名」。
- 保留一点文采，但落在动作上。

## 产品文案变自然

输入：

```text
本工具能够帮助用户高效管理任务，提高执行力，并通过智能化方式实现目标拆解。
```

改写：

```text
这个工具先不谈提高执行力。

它只帮你做一件很小的事：把一个吓人的目标，拆成今天能开始的第一步。
```

机制：

- 去掉工具介绍腔。
- 不夸大承诺。
- 把功能接到用户当下动作。

## 报告句子增强文采

输入：

```text
用户对产品价值的理解不足，导致转化效率较低。
```

改写：

```text
问题不只是用户没被说服。

是他们看完以后，还不知道这件东西该放进自己生活的哪一格。
```

机制：

- 保留判断，但减少报告腔。
- 用「生活的哪一格」让抽象转化问题可见。

## 只做诊断

输入：

```text
我们将持续赋能用户，构建完整生态，实现长期价值。
```

诊断：

```text
这句话的 AI 味主要有三处：

1. 「持续赋能」没有说明具体动作。
2. 「完整生态」是大词，读者不知道里面有什么。
3. 「长期价值」只宣布结果，没有证据和场景。

需要先补：你具体帮谁，在什么场景里，少做了哪一步，或多看见了什么。
```
