# 9 个品牌打法档案

> 来自晚点《对话 9 个消费品牌:我们从哪里找增长?》。用于类比论证——当目标品牌的处境像其中某个时,引用它的具体打法增强说服力。**只在诊断需要具体类比时按需查阅,不必通读。**

按"和谁像就抄谁的思路"使用:先看顶部速查表定位最接近的品牌,再读它的档案。

## 速查表:按处境找类比

| 如果目标品牌…… | 参考 | 它的关键打法 |
|---|---|---|
| 高端、受降级冲击小、想强化主动信任 | 之禾、CHIC JOC | 把搜索/推荐当信任入口,小红书零投放做口碑镜子 |
| 敏感肌/功效型,要把单一标签做成生态 | 薇诺娜 | 功效可视化命名 + 场景延展 + 货架沉淀复购 |
| 千元价位、要靠供应链建壁垒 | 裘真 | 自建皮料厂、报废换标准、新品表达美学 |
| 新品类、靠生活方式改变起势 | 友望、onlytree | 重新定义场景/痛点,品类词流量沉淀心智 |
| 靠稳定和克制建信任、反流量炒作 | 至本 | 十年统一定价、不用代言人、低营销干预 |
| 全球大牌要本土化、织护城河水系 | 阿迪达斯 | 产品矩阵 + 本土创新 + 消费者直接运营 |
| 踩人口/周期结构红利 | 鲜朗 | 老年猫粮赛道,品牌词跑赢品类词 |

## 之禾 ICICLE(高端女装,均价 5000+)
- 核心用户:高收入高教育女性,受降级影响有限;中间品牌竞争力下降形成"挤出效应",高端受益。
- 关键转变:高净值女性消费决策从"被动接收信息"→"主动构建信任"。
- 搜索三变化:动机从找商品变成找答案;品牌词搜索占比攀升、推荐成新客入口(搜"高智感通勤穿搭""静奢风");搜索解决"买不买"、推荐解决"知不知道"。
- 组织动作:建全渠道数字化商品库存中台,重构利益分配与业绩联动考核。

## 阿迪达斯(全球头部运动,近两年重回增长)
- 把中国看作"更庞大、多元、持续变化的结构性市场"。
- 618 靠线上强大产品矩阵而非单一爆款。
- 护城河演化:5 年前靠全球品牌势能 + 渠道覆盖;今天加上本土化产品创新 + 消费者直接运营,从"一条护城河"变成"一片交叉纵横的护城水系"。

## 薇诺娜(敏感肌护肤头部)
- 论据:2019 Meta 分析,化妆品使用人群出现敏感肌的可能性比不使用者高 7.2 倍。
- 渠道分工:天猫是确定性渠道(承载复购),小红书做深度前置蓄水(成分分析、临床数据、用户体验)。
- 命名策略:"311 屏障系列"= 神经酰胺 AP/NP + 青刺果油 + 胆甾醇 3:1:1 精准配比,功效可视化。
- 产品力证据:2021 特护霜巅峰 122 万独立用户试用,2025 仍有 120 万独立用户购买。
- 场景延展:把"敏感肌"从急救特护延展到日常护理、防晒、美白、抗老。

## CHIC JOC(国内高端女装线上头部,年销近 20 亿)
- 渠道定位:"淘宝是基本盘,直播是加速度,线下高端和国际化是品牌升维路径。"
- 小红书/抖音"基本零投放",当"用户真实反馈的镜子"而非种草工具。
- 上新一月两次,"新品是战略性输出品牌能力,不是战术性追流量窗口"。
- 坚持国际设计师合作:热爱服装、培养团队国际化视野、投入最终落到产品。
- 护城河:"你可以复制我们的某个单品、某场直播、某个渠道策略,但你复制不了一个团队十三年对产品的执着和积累。"
- 对 AI:AI 输出偏好"对称、工整、趋于平均值",有审美判断力者要求高于平均值。

## 裘真(国产千元植鞣包袋)
- 从零做植鞣皮市场教育,把小众偏好做成中高端细分品类。
- "新品最大的价值不是拉新,而是表达品牌对美学的理解。"
- 价值排序:质价比是根基,心价比自然生长;"千元定价对中国品牌意味着信任的临界点"。
- 供应链:2023 初组建自有皮料团队,累计投入 7000 多万,报废 60 多万尺、价值 2000 多万元皮料。
- 组织:自营皮革厂 + 4 家皮具工厂、团队 700 人、核心团队十年未变;新员工入职都被带到工厂。
- 升级:2025 下半年从"产品特质"延伸到"品牌人文"。

## 友望 UWANT(清洁家电新锐头部)
- 增长来自"一代人生活方式的改变"。
- 吸尘器痛点从"吸力大不大"→"倒尘杯那一刻崩溃";3 年前推首款自动集尘吸尘器,2 年从 0 做到占整体吸尘器销售额近三成。
- 除螨仪七成搜索从品类词进入(床上清洁是新场景)。
- 自我定位"0 到 1 创业品牌",统一大市场里"任何一个小品类都是一门大生意"。
- 护城河志向:做"拥有强大品牌心智的纯血中国清洁家电品牌";"好产品是'皮',其他的是'毛'"。

## onlytree(超级食物,羽衣甘蓝粉品类第一)
- 渠道分工:天猫京东做流量承载和品牌阵地,抖音小红书做前端教育种草。
- 定位:单杯 3-5 元,"比奶茶健康、比鲜榨果汁便宜、比保健品真实",是"对健康焦虑的确认""自我生活状态的低门槛投资"。
- 创新观:原料有物理上限,但"复配方案和应用场景的创新是无限的"。

## 鲜朗(国产低温烘焙猫粮开创者)
- 老年猫赛道:2018-2019 第一批养宠高峰的猫步入老年,未来 3-5 年老年猫快速增长。
- 2020-2025"烘焙粮"品类词搜索量增长数倍,"鲜朗"品牌词增速更快(品牌跑赢品类)。
- 对 AI:可提升营销运营效率,但核心配方设计仍依赖人的经验。

## 至本(国产护肤)
- 护城河:"稳定的价格体系与低营销干预一起造就的信任壁垒"。
- 全国统一定价、全年价差极小、不用代言人、不做流量炒作,价格体系近十年稳定。

## 跨品牌共识(可作为诊断的"行业共识"引用)
- **渠道**:全域不是在所有平台开店,而是让各平台做擅长的事。
- **新品**:不是越多越好,而是越"不可比"越好。
- **护城河**:9 个品牌没一个提流量能力/投放效率;答案是供应链、用户信任、好产品。
- **网红到长红**:大部分品牌上市第 3-5 年达增长峰值后下滑,跨越靠系统性组织能力。
- **AI**:帮的是效率(做图更快、投流更准、客服更高效),"下一季做什么款、品牌往哪走"仍靠人。
- **总结论**:增长不是靠突然爆发的渠道或完美投放,而是靠对产品日复一日的打磨。
