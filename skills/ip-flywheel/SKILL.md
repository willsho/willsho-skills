# IP 飞轮诊断与打法方案

## 这套方法的底层判断

IP 是这个时代最被追捧的商业资产之一——Labubu 撑起泡泡玛特两千亿市值，AI 陪伴产品把角色当核心卖点。但**几乎没有一家公司能复制迪士尼**。原因不是飞轮逻辑有多难懂，而是迪士尼的成功是**多件事必须同时成立的结果，缺任何一件，飞轮就转不起来**。

所以本 skill 的诊断不是去帮一个 IP"想几个营销点子"，而是判断：**这个 IP 的飞轮引擎是不是健康的、五个结构性前提缺了哪个、护城河是真的还是讲故事、它现在卡在哪个节点上。** 给出的建议要敢下判断、敢排优先级，而不是面面俱到的正确废话。

最锋利的一句话先放在这：**飞轮不是永动机，没有新核心 IP 持续注入，它会退化成一个"地产 + 旧 IP 授权"公司。** 引擎熄火了，飞轮终究会停下来。诊断时始终盯住"引擎还在不在烧"。

诊断**以这篇文章提炼的框架为主轴**——五节点飞轮、五个结构性前提、Seven Powers 护城河、四个自检问题。可以用任天堂、泡泡玛特、星战/漫威这些文章里出现过的例子做类比，但不要引入一整套外部 IP/营销理论（如纯粹的 4P、AARRR、品牌金字塔），那会稀释这套方法的锋利度。

## 工作流

### 第 1 步：拿到 IP 信息（主动追问缺失项）

诊断质量取决于信息质量。用户往往只丢一句"帮我看看我这个 IP 怎么做大"。**信息不足时不要硬做诊断**，先一次性列出下面 5 类关键信息中缺失的部分追问（已经给了的别重复问）：

1. **IP 类型 + 媒介** —— 是动画/虚拟角色、真人、潮玩形象、游戏角色、AI 陪伴角色还是别的？（这决定"时间无关性"——虚拟角色不老不死不塌房，真人 IP 被演员和时代绑架，是飞轮第一前提）
2. **所有权结构** —— 你 100% 拥有这个 IP 吗？和谁分权（平台/发行/原作者/代工厂）？发行/分发渠道握在谁手里？（奥斯华教训：不拥有它，一切可以在一个下午归零）
3. **商业化结构** —— 现在靠什么赚钱：主力内容（票房/订阅/付费）、周边授权、体验/线下、还是别的？各自大概占比，哪个是基本盘
4. **主力媒介的稀缺性与节奏** —— 主力内容出品频率和质量纪律如何？有没有把主力和次级媒介混在一起、过度开发（漫威/星战在 Disney+ 上就是反例）
5. **阶段 + 核心烦恼** —— 起盘几年、体量量级、现在最卡哪一环：核心 IP 不够让人爱 / 火一阵就凉 / 不拥有 IP / 不会变现 / 过度开发把自己消耗了 / 没有新 IP 接棒 / 想被投资但说不清护城河

一次性把缺的列出来问，不要挤牙膏。如果用户说"就这些，你先做"，那就**有多少用多少**，在文档里用 `(假设)` / `(待补)` 标出信息缺口，并说明补齐后结论会怎么变。

### 第 2 步：读框架，做诊断

动手前先读 `references/frameworks.md` —— 那是飞轮五节点、五个结构性前提、Seven Powers 的完整定义和文章里的真实数据。诊断时把三层框架**叠起来用**，而不是写成三段读后感：

- **五个结构性前提（体检表）**：逐项判断这个 IP 是"成立 / 部分成立 / 不成立"——①媒介的时间无关性 ②IP 100% 所有权 ③稀缺性纪律 ④长期稳定的所有权/股权结构 ⑤宇宙的情感一致性。**缺一个，飞轮就有结构性硬伤**，要直接点破是哪一个。
- **飞轮五节点（找断点）**：核心 IP → 主力媒介最大化分发 → 铺进次级媒介 → 反复货币化（不折旧）→ 体验/实体终端。判断这个 IP 走到了第几个节点、哪个节点断了、引擎（新 IP 注入）还烧不烧。这是诊断主轴。
- **Seven Powers（数护城河）**：用反定位 / 品牌 / 规模经济 / 网络经济 / 独占资源去数这个 IP 到底有"几道护城河叠在一起"。只有一道（比如只有品牌）就很脆弱；护城河是"流量能力""会投放"的，直接点破这是最不值钱的位置。

需要有说服力的类比时（"你的处境像早期迪士尼丢了奥斯华 / 像 Disney+ 上的漫威"），再去读 `references/disney-cases.md`，里面是可引用的真实案例档案。**别一上来就读 cases**，只在要引用具体打法时按需读。

### 第 3 步：产出结构化诊断文档

按下面的固定模板输出 Markdown。诊断在前、方案在后，方案必须排优先级。

## 输出模板

```markdown
# [IP / 公司名] IP 飞轮诊断与打法方案

## 〇、IP 快照
用 5-8 行交代：IP 类型与媒介、所有权结构、商业化结构、阶段与体量、核心烦恼。
有缺口的用 `(假设)` / `(待补)` 标出。

## 一、五个结构性前提体检
逐项打分（✅成立 / ⚠️部分成立 / ❌不成立），并一句话说清判断依据：
1. 媒介时间无关性：……
2. IP 100% 所有权：……
3. 稀缺性纪律：……
4. 长期稳定的所有权结构：……
5. 宇宙的情感一致性：……
> 一句话结论：致命短板是第 ___ 项，因为 ___。（缺一个飞轮就转不顺，要敢点名）

## 二、飞轮诊断 —— 你的引擎转得起来吗
沿五节点走一遍，定位走到第几个节点、断点在哪：
- 节点①核心 IP：受众是真的爱，还是只是认识？
- 节点②主力媒介分发：质量/稀缺纪律守住了吗，有没有过度开发
- 节点③次级媒介铺设：周边/授权/社群有没有反过来强化核心 IP
- 节点④反复货币化：IP 在被当一次性消耗，还是在做复利（管理出现频率）
- 节点⑤体验/实体终端：有没有不可复制的实体资产；它能不能反过来生产新 IP
> 引擎判断：新核心 IP 还在注入吗？还是在吃老本、滑向"地产+授权"？

## 三、护城河诊断 —— 别人能不能复制你
用 Seven Powers 数护城河，明确"有几道、分别是什么、哪几道是错觉"：
- 反定位 / 品牌 / 规模经济 / 网络经济 / 独占资源
> 结论：真护城河有 ___ 道；如果优势主要建立在流量/投放上，直接点破这是最脆弱的位置。

## 四、IP 打法方案（按优先级）
3-5 条具体动作，每条说清：**做什么 / 为什么（对应上面哪个诊断）/ 撬动飞轮哪个节点或补哪个前提**。
排序原则：先补最致命的结构性硬伤（通常是所有权或核心 IP 不够强），再谈飞轮提速。

## 五、关键取舍与风险
要把飞轮做起来必须放弃什么（通常是短期榨干续集的诱惑）、最大风险是什么、
哪个假设一旦不成立结论要重写。最后回答四个自检问题：
你 100% 拥有 IP 吗 / IP 有时间无关性吗 / 主力媒介保持稀缺了吗 / 你有耐心玩长期游戏吗。
```

## 几条贯穿始终的判断原则

写诊断和方案时，让这几条文章里的核心观点真正落到笔头，而不只是挂在嘴上：

- **所有权高于一切，渠道和人才都可被替代**。发行可以换、员工可以被挖走，但只要你真正拥有 IP，永远有重新开始的资本；不拥有它，一切可以在一个下午归零。这是奥斯华教训，也是诊断里优先级最高的一刀。
- **飞轮是跑出来的，不是在白板上画出来的**。迪士尼的飞轮节点（俱乐部、周边授权、金库重映）几乎全是偶然发现、快速跟进市场信号长出来的。所以方案不要给一张"完美蓝图"，而要指出现在最该抓住的那个市场信号。
- **稀缺性需要被主动管理**。次级媒介（周边/漫画/社群）可以铺满世界，但主力媒介必须低频高质、和次级媒介调性分开。一旦把主力和次级混在一起（Disney+ 上的漫威/星战），稀缺性消失，增强机制反转成消耗机制。耐心本身就是护城河。
- **让次要角色承担争议，保护核心 IP 的纯净性**。二战时唐老鸭扛枪、米老鼠远离一切战争形象。做 IP 矩阵时，用边缘角色去做脏活、做实验、做高风险投放，护住核心图腾。
- **IP 不折旧，关键是管理出现频率**。优质 IP 可以每 6-8 年（一代新观众长大）反复货币化，成本几乎为零。所以别急着榨干，永不贱卖你拥有的东西——你不知道它未来值多少。
- **核心产品必须好到配得上后面的一切**。一个"还行"的 IP 引爆不了飞轮。飞轮的前提是有一个受众真正爱上的核心 IP，这是节点①，也是所有方案的地基。
- **All-in 是飞轮公司的宿命，也是它的脆弱点**。质量上不能打折，所以必然 All-in；但 All-in 的另一面是外部环境一突变就没有缓冲。给方案时要诚实指出这层风险。
- **体验型实体资产是最深的护城河**。乐园/线下让人待得越久、花得越多，而且实体不可被复制（"你没法在 Netflix 上架一座城堡"）。如果 IP 有机会落到不可复制的实体体验上，这往往是最值钱的一步。

诊断要敢下判断、敢排优先级。用户要的是一个能拿去开会拍板、或决定投不投的东西，不是一份什么都对、什么都没说的报告。
