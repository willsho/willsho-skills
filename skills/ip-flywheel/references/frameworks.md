# IP 飞轮框架完整定义

> 来源：Acquired 播客对华特迪士尼公司的近四小时拆解（Founder Park 整理稿）。本文件是诊断时的"标准答案库"，三层框架叠起来用：先体检五个前提，再沿五节点找断点，最后用 Seven Powers 数护城河。

## 目录
1. 五个结构性前提（缺一不可）
2. 飞轮五节点（找断点）
3. Seven Powers 护城河（数护城河）
4. 四个自检问题（结尾必答）

## 一、五个结构性前提（缺一不可）

迪士尼的成功是这五件事**同时成立**的结果。诊断时逐项判断"成立 / 部分成立 / 不成立"，缺任何一个，飞轮就有结构性硬伤。

### 前提 1：媒介的时间无关性（动画/虚拟 > 真人）
动画/虚拟角色**永远不会老、不会死、不需要涨薪、永远可以工作、永远活在当下**。米老鼠 1928 年到 2026 年是同一个，没有损耗。
- 真人 IP 被演员和时代双重绑架：星战面对 Mark Hamill、Harrison Ford 变老；007 为活下来换了六七个 Bond，每次重启都有粉丝不买账。
- 经济差异：真人 IP 越成功，付给演员的薪酬越高，利润被演员分走；虚拟角色不要出场费、没有档期冲突、没有"想去拍别的片子"。
- 对照实验：1955 年迪士尼真人剧《大卫·克罗克特》红遍全美（浣熊皮帽卖 1000 万顶、周边 $3 亿），但因为绑死在 1950 年代的文化情绪上，今天没人记得；米老鼠永远是当下的。
- 一句话：**真人 IP 是"某个时代"的产物，动画/虚拟 IP 是"任何时代"的产物。** Bob Iger："As animation goes, so goes the company"，上任第一件大事就是收购 Pixar——买的是飞轮的核心引擎。
- 诊断用法：如果用户的 IP 是真人/网红/真人主播，要直接指出它先天缺这个前提，飞轮的复利天花板更低，需要靠别的前提（如强所有权 + 体验终端）补。

### 前提 2：IP 100% 所有权
迪士尼从未出售过自己的片库，把价值链关键环节一个个收回（IP 所有权 → 角色控制权 → 发行权 Buena Vista）。
- 反例对照：Universal 做哈利·波特做得不错，但不拥有全部权利，必须和 J.K. Rowling 分；很多片厂财务困难时卖了老片版权，迪士尼从不卖。
- 一句话：**不要把价值链上的关键环节交给别人。** 你不拥有它，拥有的一切可以在一个下午归零（见 disney-cases 奥斯华）。
- 诊断用法：这是优先级最高的一刀。要追问"你和谁分权""发行在谁手里"。共享 IP、平台绑定、代工厂掌握模具，都是这一项的扣分项。

### 前提 3：稀缺性纪律（质量执念 + 出现频率管理）
- **主力媒介低频高质**（电影每两三年一部），**次级媒介可铺满世界**（周边/漫画/俱乐部）。两者调性必须分开，次级不会蚕食核心，反而强化。
- **金库策略**：自有片库永不出售，每 6-8 年重映一次。
- **不过度开发**：Marvel/星战近年的麻烦，是 Disney+ 把"主力 vs 次级"的边界破坏了——剧集看起来像电影、密度极高，观众用电影标准评判每一集，稀缺性消失，增强机制反转成消耗机制。
- 一句话：**稀缺性不是自动存在的，是需要被管理的。耐心本身就是护城河。**
- 诊断用法：看主力内容的频率和质量纪律，看有没有"什么都往一个平台上堆、什么都用同一种视觉语言"。

### 前提 4：长期稳定的所有权/股权结构
迪士尼家族控制公司超过半个世纪，其他好莱坞片厂几乎每十年换一次东家。
- 飞轮需要时间建立动量——迪士尼用了 30 年才建成完整的五节点飞轮。
- 一句话：**没有人有耐心玩 30 年的复利游戏，Disney 在玩。如果你的股东每两年就想退出，这个东西永远建不成。**
- 诊断用法：问融资节奏、股东结构、创始人控制权。VC 节奏太快、急着退出，是这一项的结构性风险。

### 前提 5：宇宙的情感一致性
做个测试：你最喜欢的派拉蒙歌曲是哪首？环球影业呢？想不出来。但人人都能报出 Let It Go、A Whole New World、Under the Sea。
- 一句话：**迪士尼建的不是一堆单独的 IP，是一个有情感一致性的宇宙——所有角色、故事、歌曲都"属于同一个世界"。其他片厂有 IP，但没有宇宙。**
- 诊断用法：看用户的多个 IP/角色之间是不是互相强化、共享一个可识别的情感调性，还是各做各的、没有"同一个世界"的感觉。

## 二、飞轮五节点（找断点）

飞轮**不是 Walt 在白板上画出来再执行的，几乎是一个节点一个节点偶然长出来的**（俱乐部是影院经理的主意，周边授权是街上陌生人的灵感）。诊断时判断这个 IP 走到了第几个节点、哪个断了。

- **节点①核心 IP**：一个让受众**真正爱上**的角色/故事。一个"还行"的作品引爆不了飞轮（《白雪公主》是 All-in 赌出来的杰作）。判断：受众是真的爱，还是只是认识/路过？
- **节点②主力媒介最大化分发**：把核心 IP 在主力媒介里做到极致——高质、低频、保持稀缺。判断：质量纪律守住没有，有没有过度开发。
- **节点③铺进尽可能多的次级媒介**：周边、授权、漫画、俱乐部、社群。1934 年迪士尼周边收入就超过了电影收入（《白雪公主》还没开拍）。判断：次级媒介是在强化核心 IP，还是在廉价消耗它。
- **节点④反复货币化（IP 不折旧）**：管理出现频率，老 IP 反复重映/再卖，成本几乎为零。《冰雪奇缘》1/2/3 = 2013/2019/2027，6-8 年像钟表一样精确。判断：IP 在被当一次性消耗，还是在做复利。
- **节点⑤体验/实体终端**：主题乐园式终端，**既是消费终点，又是新 IP 的生产起点**（加勒比海盗在乐园存在 36 年后反成 $45 亿票房电影）。乐园是实体的、不可复制的。今天乐园+游轮年利润 $100 亿，是所有内容业务利润之和的两倍——飞轮上最深的护城河。

**引擎判断（最关键）**：飞轮不是永动机。**没有新核心 IP 持续注入，它会退化成"地产 + 旧 IP 授权"公司。** Walt 死后 20 年没有一部新动画超过他生前最后一部，财报还在涨但"核心在腐烂"。诊断每个 IP 时都要回答：**新核心 IP 还在烧吗？**

节点之间是**网络效应**（指数型），不是线性规模：电影让周边好卖，周边让电影更有名，两者一起让乐园更有吸引力，乐园又反过来生产新 IP。

## 三、Seven Powers 护城河（数护城河）

用 Hamilton Helmer 的 Seven Powers 框架数迪士尼的护城河，"这不是一道护城河，是五道叠在一起"。诊断时数用户的 IP 到底有几道。

- **Counterpositioning 反定位**：《白雪公主》时期愿意花三年、$150 万做动画长片——小工作室没钱做，大真人片厂赚快钱不愿做。同时拥有"米老鼠积累的品牌+资金"和"动画顶级人才"的，当时只有迪士尼一家。
- **Branding 品牌**："Walt Disney"这个名字 = 动画品质保证，消费者愿意为它多付钱（电影票、周边、门票）。
- **Scale Economies 规模经济**：砸 $600 万做一部片，让全世界付费；受众越大，单位成本越低。
- **Network Economies 网络经济**：飞轮每个节点强化其他节点；IP 消费自带社交属性（"我女儿买爱洛公主裙子，因为她朋友也有，要一起穿着玩"）。
- **Cornered Resource 独占资源**：100 年积累的 IP 库，"我没法在 5-10 年内凭空创造一个让全世界关心的 IP 宇宙"。所以 Bob Iger 的策略是直接收购（Pixar/Marvel/星战/Fox）——建不了，只能买，买了还要很多年纳入飞轮。

**诊断要点**：
- 数清楚用户的 IP 现在有几道护城河、分别是哪道、哪几道是错觉。
- **护城河里没有"流量能力""会投放"。** 如果一个 IP 的全部优势建立在"会买量""懂算法""平台扶持"上，要直接点破这是最脆弱的位置——平台规则一变就归零。
- 只有一道护城河（比如只有品牌、或只有先发）通常不够稳，要指出还缺哪几道、能不能补。

## 四、四个自检问题（结尾必答）

诊断文档结尾，用这四个问题逼用户（或投资判断）面对现实：

1. **你 100% 拥有你的 IP 吗？**（前提 2）
2. **你的 IP 有时间无关性吗？**（前提 1）
3. **你在主力媒介里保持稀缺了吗？**（前提 3）
4. **你有耐心玩长期游戏吗？**（前提 4）

飞轮需要时间建立动量，迪士尼用了 30 年。如果四个里有两个以上是"否"，要诚实告诉用户：这个 IP 很难走通迪士尼式的飞轮，要么补短板，要么换一个更现实的打法（比如靠强体验终端或强所有权单点突破，而不是奢望复制整个飞轮）。

唯一被验证过能复制这套飞轮的公司是**任天堂**：自有 IP、虚拟角色、马里奥永远不会老、严格质量控制、跨代际传承、从不贱卖角色授权（详见 disney-cases）。
